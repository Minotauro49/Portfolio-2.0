Inventory System

This is a web application built for the purpose of speeding the process the students take when borrowing videos production equipment and also to give an accurate reading through an excel sheet to the teachers regarding the status of what was left in the inventory. I built this system as a school project in the efforts to replace the slow paper version of its former self.

The Inventory System uses Bootstrap4 for its fast layout with JavaScript to enhance student’s interaction and usability. On the backend it uses PHP and Ajax for mail and dynamic page handling, Firebase API to store student data and to allow communication between excel VBA for updates on excel sheet.

AutoTrade

AutoTrade is a desktop application intentionally made to give me an advantage over my friends on ASX share market game. The main application for it was to look at all the stock data using Alpha Vantage API and gather necessary points, so I can apply my logic and algorithms to determines which stock is best to sell or buy at a specified time of day.

After the algorithm has determined the best stock, it then proceeds to log in on my ASX website account with my login credentials and trades as if I was there automatically.

I used C# for the front-end interface and Python for huddling all the HTTP requests, API calls and web scraping.
